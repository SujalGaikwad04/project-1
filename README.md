# Editkaro.in Portfolio Website

A premium, modern, responsive portfolio website built for **Editkaro.in**, a creative video editing and social media marketing agency. Designed with a dark theme, neon red accent highlight, glassmorphism cards, dynamic portfolio filtering, and fluid scroll animations.

## Tech Stack
- **HTML5**: Semantic tags, accessibility (ARIA attributes), search engine optimized (SEO) meta structure.
- **CSS3**: Custom variables, fluid responsiveness (mobile-first grids and flexbox queries), webkit custom scrollbars, complex animations (glow mesh, float, pulse, custom mouse hover-scale), dark-inversion Google map style.
- **Vanilla JavaScript**: High performance, responsive layout controls, simulated loading screen tracker, Intersection Observer scroll-reveals, filter transitions, dynamic video lightbox handling, SVG back-to-top scroll progress tracker, button click ripples, contact toast popup.

## Setup & Running
1. Since the project uses pure vanilla technology, simply open the `index.html` file in any modern web browser to view and run the website!
2. All resources (premium icons, Unsplash portraits, and YouTube video assets) are loaded via high-performance CDNs. No additional packages or dependencies are required.

## Key Features & Visual Excellence
- **Simulated Preloader**: A smooth starting screen that monitors asset loading before executing a fade-out.
- **Floating Background Glows**: Multi-colored soft blobs that drift in the background for a high-end agency aesthetic.
- **Responsive YouTube Lightbox**: Allows seamless, full-screen playback of 18 selected works across all categories. Clicking outside, using the close button, or pressing `Esc` key pauses and unloads the video instantly to save bandwidth.
- **Scroll Progress Ring**: The back-to-top button uses a circular SVG progress track that matches the user's current scroll depth.
- **Floating Labels Form**: Interactive text inputs that shift their labels into small colored badges when active or populated.
